# AI User Insight Enrichment (n8n)

This project is an automated workflow built using n8n to enrich user data with AI-based predictions.

## 🚀 Features
- Captures form submissions
- Stores data in Google Sheets
- Predicts age, gender, and nationality using APIs
- Updates enriched data in sheet
- Sends automated email with results

## 🛠 Tech Used
- n8n
- Google Sheets API
- Gmail API
- Agify API
- Genderize API
- Nationalize API

## ⚙️ Workflow
Form → n8n → Google Sheets → AI APIs → Update Sheet → Gmail

## 💡 Use Case
Useful for enriching user data with AI insights and automating analysis + notification in real-time.
