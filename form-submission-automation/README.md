# Form Submission Automation (n8n)

This project is an automated workflow built using n8n.

## 🚀 Features
- Captures form submissions
- Stores data in Google Sheets
- Sends automated email via Gmail

## 🖼️ Screenshots

## 🛠 Tech Used
- n8n
- Google Sheets API
- Gmail API
- OAuth 2.0

## ⚙️ Workflow
Form → n8n → Google Sheets → Gmail

## 💡 Use Case
Useful for collecting user data and sending instant responses automatically.
